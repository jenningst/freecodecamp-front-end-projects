API Documentation

Create User

Creates a new user object.

https://golf.api.com/user

User Attributes

  • fname

    The first name of the user.

  • lname

    The last name of the user.

  • email

    The Iemail of the user.

Update User

Updates a user object.

https://golf.api.com/user

User Attributes

  • userid

    The ID of an existing user that will receive a score in this request.

Delete User

Deletes a user object.

https://golf.api.com/user

User Attributes

  • userid

    The ID of an existing user that will receive a score in this request.

Create Score

Create a new score object.

https://golf.api.com/user

Score Attributes

  • userid

    The ID of an existing user that will receive a score in this request.

  • datetime

    The date and time when the score was received.

  • score

    The score of the user.

  • par

    The par for the course where the score was received.

Delete Score

Delete a score object.

https://golf.api.com/user

Score Attributes

  • scoreid

    The score ID for the score to be deleted.
